# Portfólio de Gustavo Bueno 

Este é o repositório do meu portfólio pessoal, uma página web interativa e responsiva desenvolvida para apresentar meus projetos, minhas habilidades e um pouco sobre minha trajetória como desenvolvedor.

# Sobre o Projeto
* Este portfólio foi criado para ser um cartão de visitas digital, combinando um design moderno e minimalista com interatividade para proporcionar uma experiência de usuário agradável e informativa. A seção de projetos utiliza Three.js para exibir meus trabalhos de uma forma visualmente cativante.

# Funcionalidades
Design Responsivo: Totalmente adaptável para uma navegação perfeita em desktops, tablets e smartphones.

Navegação Suave: Navegação com scroll suave e um menu funcional que facilita o acesso a todas as seções.

Visualização de Projetos em 3D: Uma nuvem de partículas interativa exibe meus principais projetos. Ao passar o mouse, o nome do projeto é exibido, e um clique leva diretamente ao repositório no GitHub.

Formulário de Contato Funcional: A seção de contato possui um formulário integrado com o Formspree para envio de mensagens diretamente para o meu e-mail.

Estrutura Organizada: O código é bem estruturado e comentado para facilitar a manutenção e futuras atualizações.

# Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes tecnologias:

HTML5: Para a estrutura semântica do conteúdo.

CSS3: Para estilização, layout e responsividade, utilizando Flexbox e CSS Variables.

JavaScript (ES6+): Para a interatividade da página, manipulação do DOM e a lógica do formulário de contato.

Three.js: Uma biblioteca JavaScript para criar e exibir gráficos 3D animados no navegador, utilizada na seção de projetos.

# Como Executar o Projeto Localmente
Para visualizar o projeto em sua máquina local, siga os passos abaixo:

Clone o repositório:

```
bash
git clone [https://github.com/seu-usuario/nome-do-repositorio.git](https://github.com/seu-usuario/nome-do-repositorio.git)
```
Navegue até o diretório do projeto:

```
Bash
cd nome-do-repositorio
Abra o arquivo Index.html no seu navegador: Você pode simplesmente arrastar o arquivo para a janela do seu navegador ou usar uma extensão como o "Live Server" no Visual Studio Code para uma melhor experiência de desenvolvimento.
```

E pronto! O site estará rodando localmente.
